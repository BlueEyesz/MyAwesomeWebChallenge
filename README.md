# My Awesome Web Challenge (MAWC)

Very first command: 
```css
git clone https://github.com/tsug0d/MyAwesomeWebChallenge.git
```

To play challenge, go to its directory:
```css
cd MyAwesomeWebChallenge/<challenge-name>
```

Then:
```css
docker-compose up -d
```

Access challenge via `ip:port`
